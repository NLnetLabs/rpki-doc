.. _rtrlib:

RTRlib
======

This is the user handbook of the RTRlib. It provides guidance on how to use
the library for development and gives an overview of some command line tools
that are based on RTRlib. Further information can be found on the RTRlib
website [#rtrlib_web]_ and its source code repository on Github
[#rtrlib_git]_.
